# SwitchBot BatteryDelete

Open-source mod to replace 4×AA batteries in SwitchBot smart lock with supercapacitors for reliable power.

## Files
- `kicad/` → Full KiCad project (schematics + PCB)
- `freecad/` → Parametric enclosure model (.fcstd)
- `exports/` → Ready-to-print STL files
